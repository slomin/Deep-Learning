Actual CNN:¶

Model Structure:

  • 28x28x1 (Greyscale pictures) input, normalised

  • Two convolutional layers, 32 neurons, Kernel size 3x3 and 3x3, ReLU activation

  • Max pooling (2 by 2)

  • Two convolutional layers, 64 neurons, Kernel size 3 by 3, ReLU activation

  • Another max pooling (2 by 2)

  • Flattening layer

  • Fully connected layer, 512 neurons, ReLU activation

  • Dropout 50%

  • Final fully connected layer, 10 output neurons, Softmax activation

  • The network uses Adam optimizer

In [11]:
def get_model():
    model = Sequential()
    model.add(Lambda(normalize_input, input_shape=(28,28,1)))
    model.add(Conv2D(32,(3,3), activation='relu'))
    model.add(Conv2D(32,(3,3), activation='relu'))
    model.add(MaxPooling2D(pool_size=(2, 2)))
    model.add(Conv2D(64,(3,3), activation='relu'))
    model.add(Conv2D(64,(3,3), activation='relu'))
    model.add(MaxPooling2D(pool_size=(2, 2)))
    model.add(Flatten())
    model.add(Dense(512, activation='relu'))
    model.add(Dropout(0.5))
    model.add(Dense(10, activation='softmax'))
    
    model.compile(Adam(),
                  loss='categorical_crossentropy',
                  metrics=['accuracy'])
    return model
